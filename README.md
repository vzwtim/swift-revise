# swift-revise

A starter project built with [Vite](https://vitejs.dev/), [React](https://react.dev/) and [TypeScript](https://www.typescriptlang.org/).
It includes [shadcn-ui](https://ui.shadcn.com/) components and Tailwind CSS for styling.

## Development

```sh
npm install
npm run dev
```

## Building

```sh
npm run build
```

## Linting

```sh
npm run lint
```
